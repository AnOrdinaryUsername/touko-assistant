# touko-assistant

Validate your data
```
rasa data validate
```

Train a model
```
rasa train
```

Start an action server to run [Custom Actions](https://rasa.com/docs/rasa/custom-actions)
```
rasa run actions
```

Interact with the latest model on the command line
```
rasa shell
```

## Features

- Tell jokes
- Say the weather
- Time in X place
- Print stuff using ESC/POS Thermal Printer
  - Basic text
  - Messages posted by bot
  - Checklist
- Check Calendar
- Set notification
